# HtmlToImage
利用nodejs把html生成图片返回
### 需要安装nodejs
### install
 ` npm install node-html-to-image`
 `or`
 `yarn add node-html-to-image`
 
 ### 启动
 `node NodeImage.js`
 
 ### 使用
 ![image](https://user-images.githubusercontent.com/28168503/142341531-4693b30c-ea6a-4113-9697-d6af9e238f9f.png)
 
成功后会返回该图片的地址，图片生成的路径和你html上传的路径一致
